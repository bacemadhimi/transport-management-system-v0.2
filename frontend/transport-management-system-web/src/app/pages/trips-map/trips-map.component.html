Comment Utiliser la Carte des Tournées

  • Entités colorées : Chaque entité est représentée par un cercle coloré selon son activité
  • Lignes de tournée : Cliquez sur le bouton lignes pour afficher/masquer les trajets
  • Marqueurs de tournées : Cliquez sur un marqueur pour voir les détails de la tournée
  • Filtres par niveau : Cliquez sur les boutons pour zoomer sur un niveau spécifique
  • Tableau des entités : Cliquez sur une entité pour centrer la carte
